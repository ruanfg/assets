# 基金数据接口使用说明

## 概述

本文档描述了 `EastmoneyFundRepository` 实现的基金数据获取接口，该类实现了 `FundRepository` 接口，提供从东方财富网获取基金数据的功能。

## 类结构

### EastmoneyFundRepository

基金数据仓库类，实现了 `FundRepository` 接口。

```dart
EastmoneyFundRepository({Dio? dio, FundCloudStore? cloudStore})
```

**参数:**
- `dio`: 可选的 Dio 实例，用于自定义 HTTP 客户端
- `cloudStore`: 可选的云存储实例，用于相关板块和文本解析功能

## 接口方法

### 1. searchFunds - 基金搜索

根据关键字搜索基金。

```dart
Future<List<FundSearchResult>> searchFunds(String keyword)
```

**参数:**
- `keyword`: 搜索关键字

**返回:** `List<FundSearchResult>` - 基金搜索结果列表

**示例:**
```dart
final repository = EastmoneyFundRepository();
final results = await repository.searchFunds('易方达');
for (final result in results) {
  print('${result.code} - ${result.name}');
}
```

---

### 2. fetchFundData - 获取基金数据

获取基金详细数据，包括净值、持仓、涨跌幅等信息。

```dart
Future<FundQuote> fetchFundData(String code)
```

**参数:**
- `code`: 基金代码（如 000001）

**返回:** `FundQuote` - 基金行情数据

**返回字段说明:**
| 字段 | 类型 | 说明 |
|------|------|------|
| code | String | 基金代码 |
| name | String | 基金名称 |
| dwjz | String? | 单位净值 |
| gsz | String? | 估算值 |
| gztime | String? | 估算时间 |
| jzrq | String? | 净值日期 |
| gszzl | double? | 估算涨跌 |
| zzl | double? | 增长率 |
| lastNav | String? | 上日净值 |
| yesterdayZzl | double? | 昨日涨跌 |
| yesterdayNavDelta | double? | 昨日净值变化 |
| noValuation | bool | 是否无估值 |
| holdings | List<FundHolding> | 持仓明细 |
| holdingsReportDate | String? | 持仓报告日期 |
| holdingsIsLastQuarter | bool | 是否最新季度报告 |

**示例:**
```dart
final repository = EastmoneyFundRepository();
final quote = await repository.fetchFundData('000001');
print('基金名称: ${quote.name}');
print('最新净值: ${quote.dwjz}');
print('估算涨跌: ${quote.gszzl}%');
```

### 4. fetchFundNetValue - 获取指定日期基金净值

获取指定日期的基金净值。

```dart
Future<double?> fetchFundNetValue(String code, String date)
```

**参数:**
- `code`: 基金代码
- `date`: 日期（格式：YYYY-MM-DD）

**返回:** `double?` - 净值，未找到返回 null

**示例:**
```dart
final repository = EastmoneyFundRepository();
final nav = await repository.fetchFundNetValue('000001', '2024-01-15');
if (nav != null) {
  print('2024-01-15 的净值: $nav');
}
```

---

### 5. fetchFundNetValueRange - 获取基金净值范围

获取指定日期范围内的基金净值历史。

```dart
Future<List<FundNetValuePoint>> fetchFundNetValueRange(
  String code,
  String startDate,
  String endDate,
)
```

**参数:**
- `code`: 基金代码
- `startDate`: 开始日期（格式：YYYY-MM-DD）
- `endDate`: 结束日期（格式：YYYY-MM-DD）

**返回:** `List<FundNetValuePoint>` - 净值点列表，按日期排序

**示例:**
```dart
final repository = EastmoneyFundRepository();
final navHistory = await repository.fetchFundNetValueRange(
  '000001',
  '2024-01-01',
  '2024-01-31',
);
for (final point in navHistory) {
  print('${point.date}: ${point.nav} (${point.growth ?? '-'}%)');
}
```

从指定日期开始，向后查找最近一个有数据的净值。

```dart
Future<SmartFundNetValue?> fetchSmartFundNetValue(String code, String startDate)
```

**参数:**
- `code`: 基金代码
- `startDate`: 开始日期（格式：YYYY-MM-DD）

**返回:** `SmartFundNetValue?` - 净值数据

**示例:**

```dart
final repository = EastmoneyFundRepository();

  '000001',

}
```

### 9. fetchFundHistory - 获取基金历史数据

获取基金历史净值和累计净值数据。

```dart
Future<FundHistoryResult> fetchFundHistory(
  String fundCode, {
  FundHistoryRange range = FundHistoryRange.oneMonth,
})
```

**参数:**
- `fundCode`: 基金代码
- `range`: 历史范围（默认：oneMonth）

**范围选项:**
| 选项 | 值 | 说明 |
|------|------|------|
| oneMonth | 1m | 近一个月 |
| threeMonths | 3m | 近三个月 |
| sixMonths | 6m | 近六个月 |
| oneYear | 1y | 近一年 |
| threeYears | 3y | 近三年 |
| all | all | 全部 |

**返回:** `FundHistoryResult` - 历史数据结果

**返回字段说明:**
| 字段 | 说明 |
|------|------|
| points | 历史净值点列表 |
| grandTotalSeries | 累计净值系列数据 |

**示例:**
```dart
final repository = EastmoneyFundRepository();
final history = await repository.fetchFundHistory(
  '000001',
  range: FundHistoryRange.oneYear,
);
for (final point in history.points) {
  print('${point.date}: ${point.value}');
}
```

## 使用建议

### 错误处理

所有接口方法都可能抛出异常，建议使用 try-catch 进行错误处理：

```dart
try {
  final quote = await repository.fetchFundData('000001');
  // 处理数据
} catch (e) {
  // 处理错误
  print('获取基金数据失败: $e');
}
```

### 数据缓存

对于频繁调用的接口（如基金净值查询），建议在应用层实现缓存机制以减少网络请求。

### 日期格式

所有日期参数使用 `YYYY-MM-DD` 格式：
```dart
final date = DateTime.now().toString().split(' ')[0]; // 2024-01-15
```

### 基金代码格式

基金代码通常为 6 位数字（如 000001），传入时保持原样即可，方法内部会进行 trim 处理。

## 依赖项

- `dio`: ^5.9.2 - HTTP 网络请求
- `FundCloudStore`: 可选，用于云存储相关功能
- `ShanghaiClock`: 时钟工具，用于时间处理
- `FundJsParsers`: JavaScript 解析工具

---

## 数据源

本仓库的数据主要来自以下数据源：

1. **东方财富网基金数据**: 主要基金净值、行情数据
2. **东方财富网基金搜索**: 基金搜索 API
3. **东方财富网评重数据**: 基金业绩分析数据
4. **腾讯行情**: 持仓股票实时行情
5. **云存储**: 相关板块、文本解析（需配置）

---

## 注意事项

1. 所有方法都是异步的，使用 `Future` 返回
2. 基金代码会自动 trim，无需手动处理空格
3. 部分方法需要配置 `FundCloudStore` 才能正常工作
4. 建议在 UI 层对数据进行进一步格式化处理
5. 网络请求可能失败，务必进行错误处理
